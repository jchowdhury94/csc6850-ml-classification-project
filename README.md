# Machine Learning Classification Project

## Overview

This project explores several machine learning classification techniques applied to multiple datasets. The workflow includes data preprocessing, handling missing values, dimensionality reduction using Principal Component Analysis (PCA), and training different classification models to determine which performs best for each dataset.

The implementation is provided in a Jupyter Notebook that demonstrates the full pipeline from data preparation to model evaluation.

## Repository Structure

ml-classification-project/  
├── README.md  
└── Chowdhury_Classification_Project.ipynb  

## Methods Used

The project applies common machine learning techniques including:

- Data cleaning and preprocessing
- Handling missing values through mean imputation
- Feature scaling
- Dimensionality reduction using Principal Component Analysis (PCA)
- Training and evaluating classification models such as Logistic Regression, Support Vector Machines (SVM), and Random Forest

Model performance is evaluated using accuracy, precision, recall, and F1-score.

## Dataset

The datasets used for this project are not included in this repository.

This notebook was completed as part of a course assignment. The repository is intended to showcase the project workflow, methodology, and implementation.
